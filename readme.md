# Logiciel de modification de l'arbre de décision
## Comment lancer ?
### Depuis un exécutable
Simplement télécharger l'exécutable, puis le lancer.

### Depuis le code source
1. Cloner le dépôt
2. Installer les dépendances
```sh
pip install -r requirements.txt
```
3. Lancer le script
```sh
python main.py
```
